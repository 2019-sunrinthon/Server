## Error Codes

에러 코드는 `error_type` 속성에 들어가 있습니다.

| Error Code   | description                                          |
| ------------ | ---------------------------------------------------- |
| NOT_FOUND    | 액션을 찾을 수 없습니다. 404 스펙과 동일합니다.      |
| SERVER_ERROR | 서버에서 에러가 발생했습니다. 500 스펙과 동일합니다. |
